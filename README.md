# Cavity_response_2.0

Cavity Response Simulator is designed to compute the time evolution of a chosen cavity mode in idealised cylindrical, spherical and rectangular cavities, as a response to an effective Gertsenshtein current generated by a PBH merger or early inspiral

# The library installation procedure

pyenv install 3.10.0 <br>
pyenv virtualenv 3.10.0 cavity-response <br>
pyenv activate cavity-response <br>
pip3 install -r requirements.txt
