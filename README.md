# CHASM ( Cavity Response Simulator 2.0)
Cavity response to High-frequency gravitational waves from Astrophysical Source Mergers

CHASM computes the time evolution of a chosen electromagnetic cavity mode in idealised **cylindrical**, **spherical**, and **rectangular** cavities as a response to an effective **Gertsenshtein current** generated by a PBH merger (chirp) or early inspiral. The repo also contains tools to compute **GW–cavity coupling coefficients**.

---

## What’s in this repository

The code is split into 5 main scripts:

1. **`waveform.py`**  
   Generates PBH merger waveforms (saved as `.npy` under `data/`). Source: `gw/`.

2. **`slice_integration.py`**  
   Computes the transverse slice integrals that couple to `+` and `×` GW polarisations for a chosen cavity geometry/mode and GW direction. Outputs `.npz` in `results/`. Source: `rhs/` (uses `geometry/` and `modes/` classes).

3. **`rhs.py`**  
   Builds the driving term (RHS) as a function of time using the slice integrals and a chosen waveform file in `data/`. Outputs `.npz` in `results/`. Source: `rhs/`.

4. **`ode.py`**  
   Solves the driven damped oscillator equation for the mode amplitude `c_n(t)` using an RHS file and a chosen cavity quality factor `Q`. Outputs `.npz` in `results/`. Source: `ode/`.

5. **`coupling_strength.py`**  
   Computes coupling coefficients to cavity modes as functions of GW direction (useful for early inspiral/monochromatic signals). Outputs `.npz` under `results/coupling/`. Source: `coupling/` (uses `mode` and `geometry`).

**Pipeline:** scripts 1–4 form the main “chirp/merger” pipeline; script 5 can be used independently for direction-dependent coupling studies.  
Scripts 2–5 support multiprocessing via `--nproc` (set `nproc != 1`).

## Physics summary

The mode amplitude `c_n(t)` is obtained by solving:

\[
\ddot{c}_n(t) + \frac{\omega_n}{Q}\dot{c}_n(t) + \omega_n^2 c_n(t) = -\mathcal{I}_n(t),
\]

where the driving term is computed from the effective current induced by a GW in a background magnetic field and projected onto the cavity eigenmode. Using the dependence \( \mathbf{j}_\mathrm{eff}(t,\mathbf{x}) = \mathbf{j}_\mathrm{eff}(t-x_\parallel/c)\), the 3D integral is rewritten as a 1D integral over \(x_\parallel\) with precomputed slice integrals:

\[
A_{+,\times}(x_\parallel) = \int d^2\mathbf{x}_\perp\; \mathbf{B}_{+,\times}\cdot \mathbf{E}^*_n(x_\parallel,\mathbf{x}_\perp),
\]

and then

\[
\mathcal{I}_n(t)=\int dx_\parallel\, A_+(x_\parallel)\,\partial_t^2 h_+(t-x_\parallel/c)
+ \int dx_\parallel\, A_\times(x_\parallel)\,\partial_t^2 h_\times(t-x_\parallel/c).
\]

## Quickstart (minimal working example)

1) Generate a waveform
python3 waveform.py --plot --clip

2) Precompute slice integrals for a chosen mode/geometry/direction
python3 slice_integration.py --mode-fam TM --mode-par b --mode-ind 0,1,0

3) Compute RHS using a chosen waveform file identifier in data/
python3 rhs.py --data GW_M=1.00e-06_q=1.00_20260226_142410 --mode-fam TM --mode-par b --mode-ind 0,1,0

4) Solve the mode equation for c_n(t)
python3 ode.py --data GW_M=1.00e-06_q=1.00_20260226_142410 --mode-fam TM --mode-par b --mode-ind 0,1,0
